## 意思決定構造

### デリバリーマネジメントチームの所管と境界

| 領域 | 所管 |

| コーポレート（経理・契約） | 分離済み |

### PdMチームとの境界原則

- **PdM：** 何を作るか・なぜ作るか（What/Why）の最終判断
- **PO：** どの順番で・どこまで作るか（How/When）の最終判断
- **原則：** PdMはスプリントに介入せず、POはロードマップに介入しない
