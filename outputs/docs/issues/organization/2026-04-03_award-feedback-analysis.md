## 3. 分析：何が本当の論点か

表面上は「対象範囲をどこまで広げるか」だが、**根底にある問いは2つ**。

### 問い1：Valueは誰のものか

mTG MVVとして策定されたValueを、グループ全体に適用してよいのか。

* Value原文（お客さま起点・圧倒的当事者意識・感謝と尊敬）は**mTG固有の内容ではない**。どの領域にも当てはまる行動指針

* SDチーム（片山さん）が実際にTGK担当メンバーもMVVを意識していると証言

* 一方で、公式には「myTOKYOGASの〜」と定義されているため、**名称の変更なしに全員適用すると違和感が出る**（分析者の指摘）

→ **今回の表彰を機に「グループValue」へ昇格させる**のが最もクリーンな解決策。これは制度設計の問題ではなく、キックオフでの宣言1つで済む。

### 問い2：投票制で公平性は担保できるか

人数構成上、mTG系メンバーが多数派。純粋な投票制だとTGK・EAの貢献が構造的に拾えない。

* 高松さんの懸念は正当。mTG系30名超 vs CXデジタルプラットフォームT 5名では、票数で勝負にならない

* しかし「チーム別枠」を設けるとValueの趣旨（個人の行動を評価）から外れる

→ **投票を「材料」に、GM＋TL合議で決定する方式**が落としどころ。票数だけで決めないことを明示することで、小規模チームのメンバーも候補になり得る構造にする。コメントの質・具体性を判断材料にすることで、「なぜその人か」の説得力が選考の軸になる。

## 4. 対応方針

| 論点         | 対応                                                                |
| ---------- | ----------------------------------------------------------------- |
| 対象範囲       | **及川G全員**を投票者・投票対象とする                                             |
| 票数の公平性     | 最多得票で自動決定しない。**投票＋コメントを材料にGM＋TL合議**で決定                            |
| Valueの位置づけ | キックオフで\*\*「グループValueとして位置づける」と宣言\*\*。アンケート文面は「デジタルプラットフォームＧの〜」に変更 |
| TGKメンバーの範囲 | 「FY2025年度中に当グループで活動していた方」で定義。組織表上の所属ではなく実態ベース                     |
| 異動者        | FY25中に在籍していれば対象（FY26に異動した人も含む）と明記                                 |
| キックオフでの発表  | Value表彰はグループ全体の取り組みとして位置づけ。mTG限定の文脈に置かない                          |
