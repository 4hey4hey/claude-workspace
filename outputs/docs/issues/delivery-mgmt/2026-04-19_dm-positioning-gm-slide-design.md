# DMチーム位置づけ GM向けスライド 設計テンプレ

## これは何

及川GMに「DMチームの位置づけ（モデルA）」と「貢献の見せ方（Bフレーミング＝因果チェーン）」を承認してもらうための3枚スライドの設計テンプレ。pptx-creatorで生成する前の設計フェーズ成果物。

## 3行サマリー

- DMチームを「4プロダクトの事業成果を支える条件整備の責任組織」として位置づけ、GM承認を取りに行く
- 「組織能力上げてるだけじゃん」と言われないため、因果チェーン（DMの5KR→中間効果→mTG側KR）で貢献を示す
- 承認後、プロダクト責任者4名との握りに入り、Q1末までにKR数値をセットする

---

## ゴール

及川GMに、以下2点の承認を得る：
1. DMチームの位置づけ（条件整備の責任組織、事業成果はプロダクト責任者）
2. 各プロダクトKRへの貢献の見せ方（因果チェーン）

## 聴衆

- 及川GM（承認者）
- 事業言語×具体失敗×構造問題×承認明示が刺さるパターン
- 「で、どうしたいの？」型。承認依頼形式（パターンB）で出す

## スライド構成（3枚・表紙なし）

### P1: DMチームの位置づけ（What）

| 項目 | 内容 |
|---|---|
| タイトル（主張文） | DMチームは、4プロダクトの事業成果を支える**条件整備の責任組織**として機能する |
| パンくず | デリバリーマネジメントチーム FY26方針 ／ 位置づけ |
| だから何？ | 事業成果はプロダクト責任者、デリバリーライン品質はDMが持つ、という責任分界を明示する |
| ボディ構成 | 上段：4プロダクトカード（片山TL/髙松TL/柏木TL/湯浅PdM）／下段：DMチームの共通基盤 |

### P2: 貢献の見せ方（How）

| 項目 | 内容 |
|---|---|
| タイトル（主張文） | DMの5KRは因果を通じて**myTOKYOGASの事業成果に接続**する |
| パンくず | FY26方針 ／ 貢献の構造 |
| だから何？ | DMは組織能力を上げるだけでなく、因果を通じて事業成果に効く組織であることを示す |
| ボディ構成 | 5行テーブル（DM KR → 中間効果 → mTGへの効き方 → mTG側KR） |
| 脚注 | 受発注関係を避ける運用原則（5項目）は、プロダクト責任者との握りで別途宣言 |

### P3: 承認依頼＋進め方（So What）

| 項目 | 内容 |
|---|---|
| タイトル（主張文） | この位置づけで進めることを承認いただき、**Q1末までにKR数値をセットする** |
| パンくず | FY26方針 ／ 承認依頼 |
| だから何？ | 承認後すぐに動く。握り期限と数値セット期限を明示 |
| ボディ構成 | 上部：承認依頼ブロック／中部：ロードマップ（タイムライン）／下部：想定Q&A |

## ストーリーライン（WHAT/WHY/SO WHAT/WHAT NOW）

- **WHAT**: DMチームはデリバリー機能の責任組織として機能する（P1）
- **WHY**: プロダクト責任者が事業成果、DMが条件整備、の分離で責任分界を明確化する
- **SO WHAT**: 5KRが因果を通じて各プロダクトKRに接続する（組織能力上げるだけじゃない）（P2）
- **WHAT NOW**: 承認→4月末握り→Q1末KR数値セット（P3）
