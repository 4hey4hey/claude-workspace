# Review — ドキュメント多視点レビュー

## これは何か

`issue` スキルのドキュメントレビューモードへのショートカット。
実行ロジックはすべて `issue` スキルに従う。

## 実行手順

1. `skills/issue/SKILL.md` を読み込み、**ドキュメントレビューモード**のフェーズ遷移ルールを把握する
2. Phase 1R: `skills/issue/agents/01-doc-reader.md` を読んで実行
3. Phase 2: 4つの視点エージェント（`skills/issue/agents/02-perspective-*.md`）を並列で実行
   - **必ず SKILL.md の「ドキュメントレビューモード時の読み替え」セクションを適用すること**
4. Phase 3: `skills/issue/agents/03-crosscut.md` を読んで実行
5. 出力は `skills/issue/templates/output-review.md` テンプレートに従う

## 入力

以下のいずれかで受け取る：
- `/review` のあとにファイルパスが指定されていればそのファイルを読む
- `/review` のあとにテキストが貼り付けられていればそのテキストをレビュー対象とする
- `/review` のみ（引数なし）の場合はレビュー対象を聞く

## 出力先

最終出力を保存する場合は `outputs/docs/reviews/YYYY-MM-DD_トピック.md` に置く。
