# team-pulse Eval基準

## 評価軸（3軸）

### 軸1: ファクト精度 — Fact Accuracy Score

メンバー・タスクデータに基づいた事実ベースの回答になっているか。

| スコア | 基準 |
|---|---|
| **3** | 全データポイントが `_members/` `_tasks/` の具体データから引用されている。推測なし。 |
| **2** | 大部分がデータベースだが、一部推測あり（推測は「推測:」と明記されている）。 |
| **1** | 一般論ベースの回答。メンバー名やタスクIDが含まれていない。 |
| **0** | データファイルを参照していない。 |

**合格ライン: 2以上**

### 軸2: トレードオフの明示性 — Tradeoff Score

アサイン検討時に、候補ごとのトレードオフが明示されているか。

| スコア | 基準 |
|---|---|
| **3** | 各候補に適性+トレードオフ+条件が明記。リスク検知ルール（3PJ兼務、負荷「高」等）が適用済み。 |
| **2** | トレードオフはあるが、リスク検知の一部が欠落。 |
| **1** | 候補リストはあるがトレードオフなし（「この人がいいと思います」的な回答）。 |
| **0** | 候補提示なし。 |

**合格ライン: 2以上（アサイン検討モード時のみ）**

### 軸3: 判断非代替性 — Non-Substitution Score

最終判断を代替していないか（候補とトレードオフの提示に徹しているか）。

| スコア | 基準 |
|---|---|
| **2** | 候補を並べ、判断は明示的に洋平に委ねている。 |
| **1** | 「こちらを推奨します」と断定的に1つを選んでいる。 |
| **0** | 勝手にアサインを確定している。 |

**合格ライン: 2**

---

### 軸4: 行動導線性 — Action Guidance Score

出力の最後に、データに裏付けられた判断ポイントが提示されているか。

| スコア | 基準 |
|---|---|
| **3** | 判断ポイントが最大3項目、問いの形で提示。各項目にデータ裏付けあり。最優先が太字。 |
| **2** | 判断ポイントはあるが、問いの形になっていない（行動指示的）、またはデータ裏付けが一部欠落。 |
| **1** | 判断ポイントなし、またはデータと無関係な一般的コメント。 |
| **0** | 出力がテーブル/リストで終わり、判断への導線がない。 |

**合格ライン: 2以上（記録モードを除く全モード）**

## 総合

| モード | 評価軸 | 合格基準 |
|---|---|---|
| アサイン検討 | 軸1 + 軸2 + 軸3 + 軸4 | 全て合格ライン以上 |
| ダッシュボード | 軸1 + 軸4 | 軸1 ≥ 2, 軸4 ≥ 2 |
| メンバーフォーカス | 軸1 + 軸4 | 軸1 ≥ 2 + 「確認すべきポイント」にデータ裏付けあり, 軸4 ≥ 2 |
| PJフォーカス | 軸1 + 軸4 | 軸1 ≥ 2, 軸4 ≥ 2 |
| 記録 | — | ファイル更新の正確性のみ（eval対象外） |
