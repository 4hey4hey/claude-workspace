# era Eval基準

## 評価軸（2軸）

### 軸1: 5軸評価の根拠充実度 — Evidence Score

5つの評価軸（固有データ依存度・チャネル非依存性・代替困難性・耐久性・モート）の評価に具体的根拠があるか。

| スコア | 基準 |
|---|---|
| **3** | 各軸の評価に施策固有の具体的根拠がある。agent-era-landscape.mdのデータ（定量値・予測）が引用されている。 |
| **2** | 根拠はあるが一部が一般論にとどまる。定量データ引用が不足。 |
| **1** | 評価記号（◎/○/△/×）はあるが、根拠が「〜と思われる」レベル。 |
| **0** | 5軸評価が実行されていない。 |

**合格ライン: 2以上**

### 軸2: 改善提案の具体性 — Actionability Score

判定B/C/Dの場合、「どう直せばよいか」が具体的に提案されているか。

| スコア | 基準 |
|---|---|
| **3** | API化・KPI設計・撤退基準など、実装可能な改善案が提示されている。 |
| **2** | 改善の方向性はあるが、具体的な実装イメージが不足。 |
| **1** | 「改善が必要」とだけ記述。How がない。 |
| **0** | 改善提案なし（判定Aの場合はN/A）。 |

**合格ライン: 2以上（判定A以外のとき）**
